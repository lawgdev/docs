---
title: "Signup"
api: "POST https://api.lawg.dev/v1/auth/signup"
---

<Warning>You can not be authenticated to access this route</Warning>

### Body

<ParamField body="username" type="string" required>
  The username of the user you want signup as.
</ParamField>

<ParamField body="email" type="string" required>
  The email of the user you want signup as.
</ParamField>

<ParamField body="password" type="string" required>
  The password of the user you want signup as.
</ParamField>

<ParamField body="invite" type="string" required>
  The invite code (temporary in private beta.)
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. `true` if successful, `false` if
  not.
</ResponseField>

<ResponseField name="data" type="object">

The contents of the data object

<Expandable title="Toggle object">

<ResponseField name="token" type="string">
  A generated session token. This is used to authenticate the user in future
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.lawg.dev/v1/auth/signup' \
--header 'Content-Type: application/json' \
--data-raw '{
    "username": "",
    "email": "",
    "password": "",
    "invite": ""
}'
```

</RequestExample>

  }
